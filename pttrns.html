Pttrns

Introduction

Pttrns is one of my go to sites for UI inspriation. I love how it shows interactions and has a lot of different categories to search through.

Pain Points

One of my biggest pain points with Pttrns is the mobile view. It is not responsive so I can't look at it on my phone. This is a big lose for me since I like to quickly research new mobile patterns, but now I'm required to be at my desktop. I decided to create a mobile responsive version of the website to see if I could find a way to solve this problem.

Challenge

The main challenge was to design something that worked with the existing website design. I had to find a way to design it so it would collapse when the layout would shrink.
